# 📚 AI Stock Factory Knowledge Archive Master Template Guide (v1.0)

## 🎯 개요 및 목적
본 가이드는 AI 주식 콘텐츠의 전문성과 신뢰도를 극대화하기 위해, 모든 시각 자료(슬라이드덱, 블로그 인포그래픽, 카드뉴스 등)에 통일적으로 적용되어야 하는 '지식 구조 아카이브' 디자인 시스템을 정의합니다. 핵심은 **데이터 → 해석 → 행동**으로 이어지는 명확하고 논리적인 정보 흐름을 구축하는 것입니다.

### 🎨 브랜드 스타일 가이드 재정의
*   **주요 색상 (Primary):** Deep Navy (#0A1931) — 신뢰, 깊이, 전문성 (배경, 제목 배경).
*   **강조 색상 (Accent):** Electric Cyan (#00FFFF) — 전력/AI 기술, 핵심 발견, 시선 집중.
*   **보조 색상 (Secondary):** Slate Gray (#5A6472) — 데이터 라벨링, 차트의 보조 축.
*   **폰트:** Pretendard (본문), Spoqa Han Sans Neo Bold (제목).

## 🧱 핵심 구조: The Three-Layered Flow
모든 콘텐츠는 다음 세 가지 섹션(레이어)을 순차적으로 배치하며, 이 흐름 자체가 하나의 논리적 주장입니다. 각 레이어의 비주얼 무게 배분이 달라야 합니다.

| Layer (단계) | 목적/질문 | 내용 구성 비율 (시각화) | 디자인 역할 및 강조점 |
| :--- | :--- | :--- | :--- |
| **Layer 1: 분석 근거 (Raw Data)** | "무슨 데이터가 있는가?" | 40% (차트, 원본 수치) | 객관성 제시. 차분하고 학술적인 느낌. Slate Gray/Deep Navy 중심. 출처(Source Citation) 강조. |
| **Layer 2: 해석 및 방법론 (Analysis Layer)** | "이 데이터가 무엇을 의미하는가?" | 35% (도식화, 프로세스 다이어그램) | 정책적 맥락 설명. 가장 전문성이 드러나야 하는 영역. Electric Cyan를 이용한 **흐름(Flow)** 강조 필수. |
| **Layer 3: 투자 시사점 (Actionable Insight)** | "그래서 우리는 무엇을 해야 하는가?" | 25% (핵심 수치 요약, 액션 플랜) | 결론 도출 및 행동 유도. 가장 임팩트 있고 명확해야 함. Electric Cyan와 Bold한 타이포그래피로 최종 주장 제시. |

## 📐 모듈별 상세 디자인 규칙

### A. 레이어 1: 분석 근거 (Raw Data Module)
*   **목표:** 출처 기반의 정량적 데이터를 시각화하여 신뢰도를 구축합니다.
*   **시각 요소:**
    *   **차트 유형:** Time-series Line Chart, Stacked Bar Chart (데이터 누적).
    *   **색상 사용:** 데이터 자체는 Slate Gray를 기본으로 하고, **특정 이벤트 발생 지점**만 Electric Cyan으로 마킹합니다.
    *   **레이아웃:** 좌측 상단에 데이터 출처(Source: ERCOT Report, 2024)와 시간 축을 명시하고, 차트 아래에 주요 수치를 Box 형태로 요약 제시합니다.

### B. 레이어 2: 해석 및 방법론 (Analysis Layer Module)
*   **목표:** 복잡한 정책 변화(Policy Shift)나 인프라 병목 현상(Bottleneck)을 스토리텔링으로 풀어냅니다.
*   **핵심 모듈:** **[정책 흐름 다이어그램]**
    1.  **Start Point (Trigger):** 외부 거시적 이벤트/규제 변화 발생 (Deep Navy 배경, Icon A 사용).
    2.  **Process Flow:** → '규제 겹침'이나 '병목 지점'을 화살표와 함께 명확한 순서로 연결합니다 (Electric Cyan 라인 필수).
    3.  **Output/Impact:** 해당 프로세스가 시장에 미치는 영향을 별도의 박스로 제시합니다 (Deep Navy 배경, Icon B 사용).
*   **Iconography Set 활용:** '규제', '연결', '전력 흐름'을 상징하는 5가지 아이콘 세트를 이 레이어의 주요 요소로 배치하여 시각적 전문성을 극대화합니다.

### C. 레이어 3: 투자 시사점 (Actionable Insight Module)
*   **목표:** 모든 논리를 압축하여 단 하나의 강력한 '주장'으로 마무리하고, 즉각적인 액션을 유도합니다.
*   **레이아웃 패턴:** **[The Triple Punch]** 구조 채택.
    1.  **핵심 메시지 (Headline):** 가장 큰 폰트로 최종 주장을 한 문장으로 요약 (Deep Navy 배경).
    2.  **투자 근거 3가지 (Key Pillars):** 발견된 기회/위험을 3개의 독립적인 'Pillar'로 분할하여 제시합니다. 각 Pillar는 아이콘 + 짧은 설명(Bold) + 수치적 뒷받침으로 구성됩니다.
    3.  **CTA:** "지금 바로 점검해야 할 섹터: [섹터명] (Next Step)" 문구를 Electric Cyan 박스에 강조하여 배치합니다.

## 🖥️ 포맷별 적용 가이드라인 요약
*   **유튜브 슬라이드덱 (16:9):** Layer 2와 Layer 3의 비중을 높입니다. 애니메이션 전환 효과를 통해 데이터 흐름(화살표)을 강조하며, 각 레이어 시작 시 화면 중앙에 **[AI Stock Factory] 로고 + Deep Navy 배경**으로 통일된 오프닝/클로징 슬라이드를 사용합니다.
*   **블로그 인포그래픽 (1:1 또는 4:5):** Layer 3의 'Triple Punch' 구조를 가장 크게 활용합니다. 세로 스크롤에 맞춰 레이어별 구분을 명확한 배경색 변화(Deep Navy $\rightarrow$ White/Light Gray $\rightarrow$ Deep Navy)로 처리하여 가독성을 극대화합니다.
*   **X/카드뉴스 (1:1):** 한 개의 포스팅은 단 하나의 강력한 주장(Layer 3)에 집중하며, Layer 2의 가장 핵심적인 흐름 다이어그램만 추출하여 간결하게 표현합니다.

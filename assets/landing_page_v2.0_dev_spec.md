# AI Stock Factory Playbook v1.0 Landing Page Development Specification v2.0

## 🎯 목표: 구매자 심리 단계별 전환 유도 및 규제 격차 경고 극대화
## 📐 전체 구조: Hero $\rightarrow$ Problem (Gap) $\rightarrow$ Solution (Playbook) $\rightarrow$ Trust (FAQ/Bonus) $\rightarrow$ Final CTA

### 🎨 Color Palette & Typography
*   **Primary Background:** Deep Navy (`#0A1931`)
*   **Accent / Warning:** Electric Cyan (`#00FFFF`) - **(규제 격차, 리스크 경고에만 사용)**
*   **Success / Solution:** Success Green (`#38A169`) - (Playbook 구매 후 얻는 안정감/해결책)
*   **Font Family:** Pretendard (Fallback: Noto Sans KR)

### 🖼️ Component 1: Main Banner (Hero Section)
*   **CTA Placement:** Center, Fixed-width Box.
*   **Visual Key:** 규제 격차를 상징하는 복잡하고 미스터리한 배경 다이어그램 위에 Electric Cyan의 번개/경고 신호가 지나가는 애니메이션 효과 필수.

### 📈 Component 2: Core Infographic (3-Layer Risk Analysis)
*   **Flow Logic:** Linear, 3-Step Progression (Phase 1 $\rightarrow$ Phase 2 $\rightarrow$ Phase 3).
*   **Phase 1 Styling:** Raw Data만 나열된 '혼란' 상태를 시각화. 배경에 노이즈/잡음(Noise) 효과 적용.
*   **Phase 2 Styling:** 분석 과정 자체를 보여주는 '논리적 필터' 컨셉의 다이어그램 (삼각형 플로우). **여기에 Deep Navy와 Electric Cyan을 조합하여 전문적인 느낌 극대화.**
*   **Phase 3 Styling:** Clean, Success Green으로 전환되는 '명료한 해답' 상태. 체크박스 형태의 목록 구조 사용.

### ✨ Component 3: CTA & Trust Elements
*   **CTA Button Style:** 배경색은 Deep Navy, 텍스트는 White, 그리고 버튼 주위에 Electric Cyan의 미세한 Glow 효과를 넣어 클릭을 유도.
*   **Bonus Asset Display:** 디지털 배지(Digital Badge) 형태의 모형으로 제작하여 가치를 시각적으로 '증명'할 것.

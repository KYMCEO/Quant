# 🚀 [AI Stock Factory] 규제 격차 주식 분석 마스터 스토리보드 및 비주얼 브리프

## I. 개요 및 제작 원칙
*   **최종 영상 길이:** 10~12분
*   **논리 흐름:** Pain Point (규제) $\rightarrow$ Core Data (병목) $\rightarrow$ Solution (수혜주)
*   **톤앤매너:** Deep Navy/Electric Cyan 전문성과 Warning Red의 긴급성을 결합한 '프리미엄 분석 보고서' 스타일.
*   **핵심 목표:** 시각적 구조를 통해 청중에게 **지식 습득을 통한 불안 해소 $\rightarrow$ 상품 구매 동기 부여** 경로 구축.

## II. 🎥 영상 스토리보드 및 템플릿 매핑 (총 5종)

### [Template 1] Regulatory Risk Alarm (Pain Point, 2:30분 분량)
*   **Purpose:** 규제 이벤트의 위협을 극대화하여 시청자에게 문제의 심각성을 각인.
*   **구조:** 🚨 위험 키워드 $\rightarrow$ 법적 근거 제시 $\rightarrow$ 파급 효과(경제 손실 그래프).
*   **애니메이션 포인트:** 빨간 경고 플래시, 데이터가 무너지는 듯한 애니메이션으로 긴장감 조성.

### [Template 2] Global Supply Chain Flow Map (Core Data, 3:00분 분량)
*   **Purpose:** 추상적인 규제를 물리적/지리적 병목 현상과 연결하여 전문성 확보.
*   **구조:** 전 세계 지도 기반의 흐름(Cyan Line) $\rightarrow$ 병목 구간 (Red Marker) 발생 시 데이터 레이블 노출.
*   **애니메이션 포인트:** Flow Line이 이동하며 'Gap Point'에 도달할 때마다 속도 저하 및 경고 마커 팝업.

### [Template 3] HBM Bottleneck Graph (Core Data, 2:30분 분량)
*   **Purpose:** 기술적 한계를 그래프로 증명하여 콘텐츠의 깊이를 확보.
*   **구조:** 요구 성능(TOPS 상승 곡선) vs 현재 기술 수준(Cyan 막대) vs 이상 목표(Navy 실선).
*   **애니메이션 포인트:** 간극(Gap) 부분에 빨간색 '기술적 Gap' 영역이 강조되어 노출되는 방식.

### [Template 4] Regulatory Gap Analysis Matrix (Analysis, 2:00분 분량)
*   **Purpose:** 지역별/산업별 규제 차이를 체계적으로 비교하고 투자 기회(Gap)를 발견.
*   **구조:** 매트릭스 형식 (지역 $\times$ 규제 항목). 가장 중요한 'Gap' 영역은 Warning Red로 강조.
*   **애니메이션 포인트:** 데이터가 채워지며 논리가 완성되고, 마지막에 Gap이 폭발적으로 강조되는 연출.

### [Template 5] Investment Thesis Board (Solution/CTA, 2:00분 분량)
*   **Purpose:** 분석을 바탕으로 최종 수혜주를 제시하고 행동(구매)으로 연결하는 클라이맥스.
*   **구조:** 핵심 주장 $\rightarrow$ 기업 근거 데이터 $\rightarrow$ CTA 영역 배치 (가장 크고 강력하게).
*   **애니메이션 포인트:** 모든 정보의 흐름이 중앙의 수혜주로 응집되며, 마지막 CTA 버튼에 스포트라이트를 집중.

## III. 시스템 적용 가이드라인
1.  **템플릿 재사용성:** 5가지 템플릿은 향후 제작될 모든 '규제 기반 주식 분석' 콘텐츠의 기본 구조가 되어야 합니다.
2.  **데이터 우선 원칙:** 모든 시각화는 반드시 **[Template 4]** 에서 발견된 구체적인 규제 격차(Gap)를 설명하는 근거 자료여야 합니다. 추상적 데이터 사용 금지.

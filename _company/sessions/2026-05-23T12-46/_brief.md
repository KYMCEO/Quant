# 📋 작업 브리프

**원 명령:** 코다리, Writer가 완성한 플레이북을 PDF로 변환해라.

입력: sessions/2026-05-23T10-00/playbook_draft.md
출력: data/products/playbook_v1.pdf

방법:
1. markdown을 HTML로 변환
2. Deep Navy(#0A1931) 배경, Electric Cyan(#00FFFF) 강조 CSS 적용
3. weasyprint 또는 pdfkit으로 PDF 출력
   (없으면 pip install weasyprint)

산출물 경로: data/products/playbook_v1.pdf
완료 후 파일 크기와 페이지 수 보고
